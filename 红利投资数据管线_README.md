## 详细说明

### 1. 国债收益率历史爬虫

**为什么需要？**
红利的直接竞争对手是债券。判断"红利是否便宜"不能只看股息率的历史分位，还要看国债收益率的位置。红利利差（股息率 - 国债收益率）才是真正的吸引力指标。

**需要的数据：**
- 中国10年期国债收益率（日频，10年+历史）
- 中国1年期国债收益率（日频，10年+历史）
- 中美利差（可选，辅助判断资金流向）

**数据源：**
- AKShare: `bond_zh_yield` / `bond_zh_us_rate`
- 中国债券信息网

**当前状态：** ✅ 已完成（Scraper: `Scraper/bond_yield_scraper.py`, DAO: `DAO/macro_dao.py`）

---

### 2. 红利利差/ERP计算管线

**为什么需要？**
将#1的国债收益率与已有的指数PE/股息率结合，生成两个关键指标：

$$
\text{红利利差} = \text{指数股息率} - \text{10年期国债收益率}
$$

$$
\text{ERP (股权风险溢价)} = \frac{1}{PE} - \text{10年期国债收益率}
$$

**需要的数据：**
- 10年期国债收益率（来自#1）
- 指数股息率（已有）
- 指数PE（已有）

**当前状态：** ✅ 已完成（分析模块: `analysis/spread_analysis.py`, 数据层: `DAO/macro_dao.py` + `DAO/index_dao.py`）

**数据质量验证：**
- 红利利差：仅 **19 条** 有效数据（2026年4月），因为指数估值表中的股息率字段历史数据有限 ⚠️
- ERP：**3,540 条** 有效数据（2011年~2026年），数据充足 ✅
- 当前红利利差 = **3.06%**，处于历史极低位（数据量不足，百分位参考价值有限）
- 当前 ERP = **9.78%**，历史第 56 分位（中等）
- **结论：利差指标受股息率数据限制，需要寻找更好的股息率数据源**

### 3. 行业估值拆分管线

**为什么需要？**
红利指数的成分股集中在5-6个行业。如果某个行业估值泡沫化，会拉高整个指数的估值，而其他行业可能仍便宜。

已覆盖的行业指数：
- ✅ 中证银行(399986) — 行情2662条, 估值20条
- ✅ 中证公用(000966) — 行情3962条, 估值20条
- ✅ 中证煤炭(399998) — 行情3963条, 估值20条
- ✅ 中证消费(000932) — 行情3963条, 估值20条
- ✅ 中证交通运输(H30171) — 行情600条, 估值20条

注意：估值数据（PE/股息率）当前只有近期约20条，历史数据有限。

**数据源：**
- AKShare: `index_zh_a_hist` 拉取各行业指数日线
- 中证指数官网 xls 估值文件

**当前状态：** ✅ 已完成（含行情+估值爬虫管线，代码: `Scraper/loader.py` + `Scraper/_akshare.py`）

### 4. ETF份额与折溢价历史

**为什么需要？**
- **份额变化** = 聪明钱的流向。份额持续增长说明资金在持续流入红利风格。
- **折溢价** = 买贵了还是便宜了。折价买入相当于多赚一点。
- 份额下降 + 溢价 = 警惕信号

**需要的数据：**
- 515450 ETF每日份额（万得/AKShare/天天基金）
- 515450 每日净值 vs 收盘价（折溢价率）
- 同类ETF（512890红利低波ETF华泰）份额对比

**数据源：**
- AKShare: `fund_etf_fund_info_em` / `fund_etf_spot_em`
- 天天基金网

**当前状态：** ❌ 未实现

### 5. 资金流向数据管线

**为什么需要？**
北向资金（外资）是国内红利板块的重要边际定价者。外资持续流入时，红利估值有支撑；外资流出时，红利承压。

**需要的数据：**
- 北向资金/陆股通每日净流入（总量 + 红利成分股拆分）
- 两融余额（市场杠杆水平，高杠杆意味着风险偏好高，不利于红利）
- 南向资金（港股红利参考）

**数据源：**
- AKShare: `stock_hsgt_north_net_flow_in_em` / `stock_margin_detail_szse`
- 东方财富

**当前状态：** ❌ 未实现

### 6. 宏观指标采集管线

**为什么需要？**
经济周期位置决定了市场风格偏好：

| 指标 | 对红利的影响 |
|------|-------------|
| PMI < 50 → 防御偏好↑ → 红利相对受益 |
| PPI ↓ → 上游利润承压 → 煤炭/资源类红利承压 |
| CPI 低位 → 宽松预期 → 有利高股息资产 |
| 社融↑ → 经济复苏 → 风格可能切换至成长 |

**需要的数据：**
- PMI（月频，10年+）
- CPI / PPI（月频，10年+）
- 社融增速（月频，10年+）
- M2增速（月频，10年+）

**数据源：**
- AKShare: `macro_china_pmi` / `macro_china_cpi_yearly`
- 国家统计局官网

**当前状态：** ❌ 未实现

## 项目架构

```
Scraper/                    ← 数据源层
  loader.py                 ← 对外数据加载管线
  _akshare.py               ← AKShare 私有封装
  base_scraper.py           ← HTTP 爬虫基类（旧）
  scrapers/                 ← 旧式爬虫（逐步迁移至loader）

DAO/                        ← 存储层
  index_dao.py              ← 指数行情+估值
  macro_dao.py              ← 宏观/债券
  options_dao.py            ← 期权

analysis/                   ← 衍生计算层
  spread_analysis.py        ← 利差/ERP分析

数据流：
  loader.py → _akshare.py → 原始 DataFrame
           → 清洗转 Record
           → DAO upsert 写入 DB
```
